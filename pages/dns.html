Conocimiento DNS

Todo lo que necesitas saber sobre el Domain Name System resumido en una tabla.

Pregunta Respuesta
¿Qué es un nombre de dominio? Un nombre de dominio es la dirección fácil de recordar que identifica un sitio o servicio en Internet, por ejemplo: google.com. Este nombre se traduce a una dirección IP mediante DNS.
¿Qué es un servidor DNS? Un servidor DNS es un servidor encargado de traducir nombres de dominio a direcciones IP y viceversa.
¿Cuál es la función principal del DNS en una red? La función principal del DNS es convertir nombres de dominio legibles para humanos en direcciones IP que las computadoras utilizan para comunicarse.
¿Qué problema existiría si no existiera el DNS? Sin DNS, las personas tendrían que memorizar direcciones IP para acceder a sitios web, lo que haría Internet mucho más difícil de usar.
Explique todos los pasos que sigue el DNS para resolver un nombre.
  1. El usuario escribe un dominio en el navegador.
  2. El sistema revisa la caché local.
  3. Si no encuentra la IP, consulta al servidor DNS configurado.
  4. El DNS consulta servidores raíz.
  5. Luego consulta servidores TLD (.com, .mx, .org, etc.).
  6. Después consulta el servidor autoritativo del dominio.
  7. El servidor devuelve la dirección IP.
  8. El navegador se conecta al servidor usando esa IP.
¿Qué es la ICANN y cuál es su función respecto al DNS? ICANN significa "Internet Corporation for Assigned Names and Numbers". Es una organización internacional cuya función es coordinar y administrar el sistema global de nombres de dominio y direcciones IP para garantizar que Internet funcione correctamente.
¿Cuáles son los ataques más comunes a un DNS?
  • DNS Spoofing
  • DNS Cache Poisoning
  • Ataques DDoS
  • DNS Hijacking
  • Túneles DNS
¿Qué puerto utiliza DNS por defecto? DNS utiliza por defecto el puerto 53.
¿Qué protocolos utiliza DNS para funcionar? DNS funciona principalmente con los protocolos UDP y TCP.
¿Qué es un servidor DNS primario? Un servidor DNS primario es el servidor principal que almacena y administra los registros originales de una zona DNS.
¿Qué es un servidor DNS secundario? Un servidor DNS secundario es un servidor de respaldo que mantiene una copia de la información del servidor primario.
¿Qué es la caché DNS? La caché DNS es el almacenamiento temporal de consultas DNS para acelerar futuras búsquedas.
¿Qué comando se utiliza en Windows para consultar información DNS? En Windows se utiliza el comando nslookup.
¿Qué comando equivalente se utiliza en Linux? En Linux el comando equivalente más utilizado es dig.
¿Qué hace el comando nslookup? nslookup permite consultar información DNS, como obtener la IP asociada a un dominio.
¿Qué hace el comando dig? dig permite realizar consultas DNS detalladas y mostrar información técnica sobre la resolución de nombres.
¿Qué diferencias existen entre DNS público y DNS privado?
  • DNS público: accesible para cualquier usuario en Internet.
  • DNS privado: utilizado dentro de redes internas y no accesible públicamente.
Menciona dos servidores DNS públicos populares. Google DNS → 8.8.8.8 Cloudflare DNS → 1.1.1.1
¿Cuáles son las reglas para elegir un nombre de dominio?
  • Debe ser único.
  • Fácil de recordar.
  • Corto y claro.
  • Sin espacios.
  • Puede usar letras, números y guiones.
  • Debe incluir una extensión válida (.com, .org, .mx, etc.).
¿Cómo se registra un nombre de dominio? Un dominio se registra mediante una empresa registradora autorizada.
¿Cuál es el costo de un dominio web y por cuánto tiempo? El costo normalmente varía entre 10 y 50 USD al año, dependiendo de la extensión y el proveedor. Generalmente se registra por 1 año, aunque puede pagarse por varios años.
¿Cuál es el proceso y los requisitos para adquirir uno?
  1. Elegir un nombre disponible.
  2. Seleccionar una extensión (.com, .net, etc.).
  3. Registrarse en una empresa registradora.
  4. Proporcionar datos personales y de contacto.
  5. Realizar el pago.
  6. Configurar los servidores DNS del dominio.
